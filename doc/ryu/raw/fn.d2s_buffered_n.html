pub unsafe fn d2s_buffered_n(f: f64, result: *mut u8) -> usize
Expand description

Print f64 to the given buffer and return number of bytes written. Ryū’s original formatting.

At most 24 bytes will be written.

Special cases

This function represents any NaN as NaN, positive infinity as Infinity, and negative infinity as -Infinity.

Safety

The result pointer argument must point to sufficiently many writable bytes to hold Ryū’s representation of f.

Example

let f = 1.234f64;

unsafe {
    let mut buffer: [u8; 24] = std::mem::uninitialized();
    let n = ryu::raw::d2s_buffered_n(f, &mut buffer[0]);
    let s = std::str::from_utf8_unchecked(&buffer[..n]);
    assert_eq!(s, "1.234E0");
}
